# LadyNuggets HQ Prompt System v1 (2026)

## Goal
Generate premium anime/ecchi images with consistent quality for exclusives.

## Prompt blocks (always)
1. Character: age-appropriate anime woman + hair/eyes + expression
2. Outfit: concrete garments and accessories
3. Pose: camera-friendly and readable
4. Camera: close-up / medium / full body + lens style
5. Lighting: soft cinematic + rim light
6. Scene: simple but rich context
7. Quality tags: masterpiece, best quality, ultra detailed, sharp focus

## Negative block (always)
lowres, blurry, bad anatomy, bad hands, extra fingers, malformed limbs, watermark, text, logo, jpeg artifacts

## Copy rules (for selling)
- No generic “I made this” description.
- Format:
  - Hook (1 line)
  - Fantasy/context (1-2 lines)
  - Offer line: exclusive drop $10
  - CTA line: DM to claim
- Tags for DA: 12-20 real tags (without #)

## Classification gates
- Public channels: allow SFW + ECCHI
- Explicit NSFW: reject for public, keep for exclusive private pipeline only
- Minimum quality to pass: 7/10
