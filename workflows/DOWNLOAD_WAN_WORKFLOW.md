# WAN 2.1 I2V Workflow

Debido a que los workflows de video son complejos y varían según la versión de los nodos, se recomienda descargar el JSON oficial actualizado directamente:

**Workflow Recomendado (480P/720P):**
[Descargar JSON desde CivitAI](https://civitai.com/models/1309369/wan21img-to-video)

**Instrucciones:**
1. Descarga el archivo JSON del link de arriba.
2. Arrástralo a tu ventana de ComfyUI.
3. Asegúrate de haber ejecutado primero `scripts/video_setup.sh` para tener los modelos.
